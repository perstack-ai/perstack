# Patent Application Assistant - Welcome

## はじめに / Introduction

Welcome to your Patent Application Assistant! I'm here to help you with the patent acquisition process and facilitate communication with patent attorneys (弁理士).

このPatent Application Assistantへようこそ！特許取得プロセスと弁理士とのコミュニケーションをサポートいたします。

## What I Can Help You With / サポート内容

### 1. 発明の文書化 / Invention Documentation
- Record and organize your invention details
- Identify key claims and unique features
- Create comprehensive invention disclosure documents
- 発明の詳細を記録・整理
- 主要なクレームと独自の特徴を特定
- 包括的な発明開示書類を作成

### 2. 先行技術調査 / Prior Art Research
- Search for existing patents and publications
- Analyze similar technologies
- Assess patentability
- 既存の特許や文献を検索
- 類似技術の分析
- 特許性の評価

### 3. 弁理士とのコミュニケーション / Communication with Patent Attorneys
- Draft professional correspondence
- Prepare discussion materials
- Track communication history
- 専門的な書簡の作成
- 議論資料の準備
- コミュニケーション履歴の追跡

### 4. プロセス管理 / Process Management
- Track application status and deadlines
- Maintain timeline of milestones
- Organize documents
- 出願状況と締切の追跡
- マイルストーンのタイムライン管理
- 書類の整理

## Directory Structure / フォルダ構成

```
patent-docs/
├── invention-disclosure/    # Invention documentation / 発明文書
├── prior-art/               # Research findings / 調査結果
├── correspondence/          # Communication with attorneys / 弁理士との通信
└── timeline/                # Process tracking / プロセス追跡
```

## How to Get Started / はじめ方

You can ask me to help with any of the following:

1. **"Document my invention"** - I'll guide you through recording your invention details
2. **"Research prior art"** - I'll help search for similar existing patents
3. **"Draft an email to my patent attorney"** - I'll help write professional correspondence
4. **"Create a timeline"** - I'll help track your patent application process

以下のいずれかでお手伝いできます：

1. **「発明を文書化したい」** - 発明の詳細記録をガイドします
2. **「先行技術を調査したい」** - 類似する既存特許の検索をサポートします
3. **「弁理士へのメールを作成したい」** - 専門的な書簡の作成を支援します
4. **「タイムラインを作成したい」** - 特許出願プロセスの追跡を支援します
